## Run Locally

Clone the project

```bash
  git clone https://github.com/Haider-Mukhtar/Haider-Mukhtar-Portfolio-V2.git
```

Go to the project directory

```bash
  cd Haider-Mukhtar-Portfolio-V2
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
